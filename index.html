Native Feel
Under-the-hood matrix multiplications for absolute positioning offer 3D support and provides 60fps granular animations.
Integrated Animations
Animations gives visual clues to the audience with subconscious understanding of how to interact with an application. They are essential to user experience and built-in to the framework. No plugins or CSS magic needed.
Standard Syntax
The absence of JSX or templating language can seem foreign as almost every other framework is using their own custom language. We use standard ES6 syntax all the way to ensure native browser compatibility.
Efficient Update Algorithm
Views are state machines that perform partial re-renders for change in declared state operations. Syntax is straight-forward and there is only one source of truth of view state data.
Abstracted Layout Engine
While CSS is still used for styling of the content, whereas positioning and sizing is handled in a more pragmatic way. Arva abstracts away some of the concerns that many front-end developers face, which includes CSS deep-dives and complex layout directives.
More than just Visuals
Routers and controllers offer support for top-level control and data manipulation.

What is Arva?
How can I contribute to Arva?
Good question! First thing to do is to head over to our repository. Then, get into the code and make a pull request. If you have any questions about how the code should work or style guidelines, make an issue on the repository or contact Karl
How did Arva originate?
Arva started in 2015 when seeing the need for developing high-quality mobile apps on par with the native equivalents. At the time, Facebook had recently split their mobile app into two in order to achieve satisfactory results. Today, there is still a need for a reliable option to make hybrid worth it and Facebook (and many others) are still using native apps.
How many ARVA apps exist right now?
In total 22 arva applications have been made, 17 of which are in active use at this moment. Over half of the applications have bigscreen optimizations or full fledged desktop management applications.
Does Arva UI Kit include component X?
Components for most use-cases are included out of the box. If you have requests of your missing favorite component, make an issue, or send us a pull-request.
Is Arva an acronym?
No! Arva is just the name of the product. It's proven to be sticky to peoples mind and the latin-sounding name goes well for many different speakers in other parts of the world. If you really want, we invented a backronym: Assemble Rapid Velocity Applications.

Roadmap
Who is building Arva?
Arva is maintained and designed by Karl Lundfall at Bizboard. The code started as a fork off the library Famous and has realized the promises of that framework and combined it with modern paradigms of today's front end tooling.
What is the bigger picture of Arva?
Arva Foundation is the core of the Arva User Experience Platform where developers have complete freedom to hack their creations. It's a part of a suite of products that are all aimed at simplifying application development. Read more here.
What is the future of Arva?
Arva as a Javascript framework is under constant development and maintenance. Closest ahead is to improve the current routing logic to be more flexible. We'll continuously make performance bumps to make rendering smoother. We'll also make improved solutions for external database integration.
